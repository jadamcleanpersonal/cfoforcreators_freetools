# CreatorLand Outreach Tracker — Multi-Week Queue

**Targeting philosophy:** professional creators in ANY niche (not finance creators). Creators with multi-tag role profiles like YouTuber + Creator + Influencer, who run content as a real business but probably don't have great finance systems.

**Source filter:** `https://www.creatorland.com/discovery/people?creatorType=YouTuber,Podcaster,Influencer` — Creator Type = YouTuber + Podcaster + Influencer. CreatorLand reports **814 pages** of results in this filter, ~50 creators per page = ~40,000 total. We're sweeping the first 3 pages here.

**Sweep date:** June 22, 2026. ~150 creators captured below.

**Cap reminder from `post_launch_ops.md` Channel 5:** 10–20 DMs/week max. Working from this queue at that pace = 7–10 weeks of material. Adding to this list with more sweeps gives you 4+ months of runway.

## Critical workflow notes

- CreatorLand has **no audience-size filter**. You must click each profile to verify the number.
- Many profiles show **Audience size: 0** (creator never connected platforms) — skip those.
- Target: **10k–100k audience**. Skip <10k or >250k.
- Profile URL format: `https://www.creatorland.com/[handle]` — handles already captured below so you don't need to search.

## How to extend this list

The sweep below covers pages 1-3 of the YouTuber+Podcaster+Influencer filter. To add more candidates, navigate directly to higher pages:

```
https://www.creatorland.com/discovery/people?creatorType=YouTuber%2CPodcaster%2CInfluencer&page=4
https://www.creatorland.com/discovery/people?creatorType=YouTuber%2CPodcaster%2CInfluencer&page=5
...up to page 814
```

Or change filter combos to cast different nets:
- `?creatorType=YouTuber%2CPodcaster` (drops Influencer — more focused on long-form creators)
- `?creatorType=Streamer%2CYouTuber` (twitch + YT crossover)
- `?contentType=Beauty%20%26%20Makeup` (beauty creators across all role types)
- `?contentType=Gaming` (gaming creators)
- `?contentType=Travel` (travel creators)
- `?contentType=Food%20%26%20Drink` (food creators)

Each filter returns hundreds-to-thousands of pages. You can build a 1000+ creator queue across 4-5 filter combos × 5-10 pages each.

## Self-Service Sweep Workflow (extend the list yourself)

When you've worked through the 150 above and want more, run more sweeps yourself in 10-min batches:

### Step 1: Pick a filter combo

Same Creator Type filter, different page:
- Page 4: `https://www.creatorland.com/discovery/people?creatorType=YouTuber%2CPodcaster%2CInfluencer&page=4`
- Pages 5, 6, 7... (just change the `page=N` at the end)

Or different filters:
- YouTuber only (largest pool): `?creatorType=YouTuber`
- Podcaster only (sponsor-rate gold): `?creatorType=Podcaster`
- Streamer + YouTuber (Twitch+YT): `?creatorType=Streamer%2CYouTuber`
- Beauty content: `?contentType=Beauty%20%26%20Makeup`
- Gaming: `?contentType=Gaming`
- Travel: `?contentType=Travel`
- Food: `?contentType=Food%20%26%20Drink`
- Fitness: `?contentType=Fitness`

### Step 2: Open Chrome DevTools (F12) → Console tab → paste this:

```js
copy([...document.querySelectorAll('a[href^="/"]')]
  .filter(a => a.querySelector('h2,h3,h4'))
  .map(a => {
    const name = a.querySelector('h2,h3,h4')?.innerText;
    const tags = a.querySelector('span,div')?.innerText;
    const handle = a.getAttribute('href');
    return `| ${name} | https://www.creatorland.com${handle} | ${tags} | tax | A |`;
  })
  .join('\n'));
```

This copies a markdown table row for each creator on the page directly to your clipboard. Paste into your tracker.

### Step 3: Verify audience + send

Click each creator's profile URL, verify their audience is 10k–100k, send the matched DM template, log in your sheet.

Use the matched letter from the priority table. Personalize ONLY the first line with the specific thing you saw on their profile/content. Voice: lowercase, peer-to-peer, "i built it because [creator pain], sharing in case useful." No introductions, no paid product mention, no asks.

## Weekly rhythm (from post_launch_ops.md Channel 5)

- Cap: 15–20 DMs per week. Don't push past 20.
- Time: Monday PM, 90 min block. 8-10 min per DM (most of which is the profile click + personalization read).
- Friday: review the week — what landed, what didn't, adjust which roles you target next week.

At 15-20 DMs/week, this 150-creator queue is 7-10 weeks of material before you need to extend it.
